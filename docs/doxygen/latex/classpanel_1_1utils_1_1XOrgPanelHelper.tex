Sets up the status bar for the X11/\+Xorg session. 

This function sets the frameless window hint for the target status bar window and anchors it to top-\/left of primary screen. After showing the window, it sets the {\ttfamily \+\_\+\+NET\+\_\+\+WM\+\_\+\+WINDOW\+\_\+\+TYPE\+\_\+\+DOCK} and {\ttfamily \+\_\+\+NET\+\_\+\+WM\+\_\+\+STRUT\+\_\+\+PARTIAL} to reserve the space needed. \begin{DoxyNote}{Note}
This function also ensures that the status bar is avaliable on all the virtual desktop.
