panel::backend::VolumeControl::VolumeControl ( QObject * parent = nullptr)
explicit

The constructor for VolumeControl.

Initializes the PulseAudio threaded main loop and context, sets up the state callback, and starts the PA thread. Subscribes to all sink, source, sink-input, source-output, and server events once the context reaches PA_CONTEXT_READY.
