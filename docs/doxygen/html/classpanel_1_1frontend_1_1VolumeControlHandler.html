◆ GetVolumeIcon()

QString panel::frontend::VolumeControlHandler::GetVolumeIcon ( ) const

Returns the material icon name reflecting the state of the default audio output sink.

Note
Icon mapping reflects the active (default) output device: muted → volume_mute 0% → volume_off 1 – 50% → volume_down 51 – 100% → volume_up
Returns
(QString) One of: volume_mute, volume_off, volume_down, volume_up.
