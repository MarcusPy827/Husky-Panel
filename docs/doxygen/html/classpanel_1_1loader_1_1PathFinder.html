QString panel::loader::PathFinder::GetConfigBaseDir ( )
static

Gets the base directory for configuration files.

This function first gets the target path, it will create the directory if it doesn't exist, and then return the path.

Note
This function relies on QStandardPaths::writableLocation.
The program crashes if the config directory failed to create.
See also
GetSystemConfigDir().
Returns
(QString) The base directory for configuration files.
